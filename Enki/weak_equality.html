explanation

It would be nice to know what each of these escaped characters does:

  • "\t" is horizontal tab
  • "\r" is carriage return
  • "\n" is line feed

Weak equality operator ( == ) compares the value of the two operands. When a number and a string are compared using ==', JavaScript tries to convert the string into a number and compare them as numbers. The numerical value of a string containing only white space (which all of the above mentioned escaped characters are) is0', which is equal to 0'. Therefore, the result of the comparison istrue'.
